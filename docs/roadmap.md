## Decision Network v2 Ideas

### Ambient Anchor Nodes
- Every 15–20 seconds, randomly select 2–3 nodes as temporary anchors.
- Anchors drift very slowly while nearby nodes continue moving normally.
- This should create the impression of stable decision points without distracting the user.
- Re-evaluate after the landing page and simulation engine are complete.
